HTML (HyperText Markup Language)

What is HTML?

  • HTML stands for HyperText Markup Language
  • Used to create and structure web pages
  • Uses tags (within <>) to mark parts of content on a browser

Basic Structure of HTML

PartMeaning
<!DOCTYPE html>Document type declaration
<html>Root of HTML document
<head>Page info (not visible to users)
<title>Title in browser tab
<body>Page content visible to users
<!DOCTYPE html>
<html>
<head>
  <title>My First Page</title>
</head>
<body>
  Hello, World!
</body>
</html>

HTML Tags

  • Tags are keywords in <>
  • Most tags have an opening and closing tag
  • Example: <p>This is a paragraph</p>

Types of HTML Elements

  • Block-level Elements: Start on new line, take full width (div, h1-h6, p, ul, ol, li, table)
  • Inline Elements: No new line, take only needed width (span, a, img, b, i)

Text Formatting Tags

<b>: Bold
<i>: Italic
<u>: Underline
<strong>: Important
<em>: Emphasized
<mark>: Highlight
<small>: Small text
<br>: Line break
<hr>: Horizontal line

Attributes

  • id: Unique identifier
  • class: Grouping for CSS or scripts
  • src: Source file (images, audio, video)
  • href: Link target (anchor, CSS, JS)
  • width, height: Size for images, tables, etc.
